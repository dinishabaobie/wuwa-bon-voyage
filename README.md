# 泰提斯终端 · 鸣潮观测档案

> 群星静默，直至找到它们的路。

一个以《鸣潮》世界观为主题的非官方沉浸式资料网站。网站借用黑海岸与泰缇斯终端的叙事视角，将角色档案、剧情解析、版本记录与索拉里斯世界纪年整理成可交互的观测界面。

**在线访问：** [进入泰提斯终端](https://dinishabaobie.github.io/wuwa-bon-voyage/)

## 主要内容

- **首页观测**：沿今州、黎那汐塔、拉海洛等区域展开剧情与世界观记录。
- **观测对象**：收录角色档案、身份信息与相关观测记录。
- **观潮**：
  - **航程纪年**——按现实版本梳理《鸣潮》的更新历程；
  - **溯洄纪年**——按游戏内先后顺序整理索拉里斯的重要事件；
  - **深度推演**——展开重点篇章的剧情主题与因果链。
- **群像**：以交互关系图呈现角色之间的联系。

## 设计与技术

网站以深海、星图、黑花与数据终端为主要视觉语言，并针对桌面端和移动端分别优化交互体验。

- Vite
- Vanilla JavaScript
- GSAP / ScrollTrigger
- Three.js
- Lenis
- Playwright

```bash
npm run dev
```

```bash

npm run build   # 生成生产版本

```

## 项目结构

```text
src/
├── chronology.js  # 观潮 · 游戏内事件时间轴
├── timeline.js    # 观潮 · 版本时间轴
├── tide.js        # 观潮页面与交互
├── observation.js # 观测对象
├── relation.js    # 群像关系图
└── main.js        # 首页章节与动效
```
